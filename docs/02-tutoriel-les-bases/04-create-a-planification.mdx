# 4. Créer une planification

:::info La planification

Une planification,  C'est le **lien entre le coach et l'élève**.
C'est un emploi du temps, un guide, qui comporte des plans de nutrition, des programmes d'entraînement, des instructions de repos et de bilans **au jour le jour**. <br />
C'est l'outil de communication entre le coach et l'élève.

:::
