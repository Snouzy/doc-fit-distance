## 1.3.0
- Général :
    - Ajout d'un chat avec une personne réelle en direct sur chaque page de l'application (en bas à droite)
- Page détail d'un élève : 
    - Ajout d'un onglet `Résumé`. On y retrouve toutes les informations
    importantes de l'élève rentrées initialement par celui-ci :

        - Allergies
        - Pathologies
        - Travail (sédentarité)
        - Antécédents sportifs
        - Type de corps (ectomorphe, endomorphe, mésomorphe)
        - Bonus : Pourquoi avoir fait appel à un coach
